# AgentLink

AgentLink is een open platform voor AI agents. Deze repository bevat fase 0 van de codebase: projectstructuur, configuratie en documentatie.

## Tech stack

- Next.js (App Router)
- TypeScript (strict)
- Prisma + PostgreSQL
- Tailwind CSS
- NextAuth
- Vitest + Playwright

## Starten

```bash
pnpm install
pnpm dev
```

## Huidige status

- Fase 0 skelet staat klaar
- Geen features geimplementeerd
- Geen database modellen toegevoegd
