# src

Doel: alle applicatiecode voor AgentLink.

## Overzicht
De `src` map bevat app routes, componenten, gedeelde libraries, types en styles.

## Bestanden
- `app/`: Next.js App Router
- `components/`: herbruikbare UI en domeincomponenten
- `lib/`: gedeelde utilities en clients
- `types/`: TypeScript type definities
- `styles/`: globale styles
- `info_src.md`: documentatie van de src map

## Afhankelijkheden
- Gebruikt door: Next.js runtime
- Hangt af van: root config bestanden en dependencies

## Patronen
- Strikte scheiding tussen routes, UI, en infrastructuurcode
